Artistic Style Automatic Code Formatter
==============================================

To build on Linux:
-------------
Download the latest stable version from https://sourceforge.net/projects/astyle/files/astyle/

mkdir out
cd out
cmake ../
make

To build on Windows:
----------
you may just download a pre-built Windows executable from https://sourceforge.net/projects/astyle/files/

For more info
------------------
http://astyle.sourceforge.net/
